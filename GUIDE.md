# H5 RSS 阅读器 - 使用指南

## 🎉 欢迎使用

恭喜你!H5 RSS 阅读器已经成功创建。这是一个功能完整、界面优美的移动端 RSS 阅读应用。

## 📱 立即体验

开发服务器已经在运行中，点击预览按钮即可查看应用!

**访问地址**: http://localhost:5174

## 🚀 快速上手

### 第一步：添加 RSS 源

1. 点击首页右下角的蓝色"+"按钮
2. 输入 RSS/Atom 订阅地址
   - 示例：`https://rsshub.app/...`
   - 或者你喜欢的博客、新闻网站 RSS 地址
3. 系统会自动解析并显示预览
4. 点击"添加"按钮完成添加

### 第二步：浏览文章

1. 在首页点击任意 RSS 源卡片
2. 进入文章列表页面 (海报式布局)
3. 点击文章卡片查看详细内容

### 第三步：个性化设置

1. 在阅读页面点击右上角字体图标
2. 调整字体大小 (14-24px)
3. 调整行间距 (1.5-2.5)
4. 切换主题 (日间/夜间/复古)

## 💾 数据备份

### 导出数据
1. 进入设置页面
2. 点击"导出 RSS 源"
3. 浏览器会下载 JSON 备份文件
4. 建议定期备份

### 导入数据
1. 进入设置页面
2. 点击"导入 RSS 源"
3. 选择之前下载的 JSON 文件
4. 系统自动恢复所有数据

## ⚠️ 注意事项

### CORS 限制
由于浏览器同源策略，部分 RSS 源可能无法直接访问。如果遇到解析失败:
- 检查网络连接
- 更换 RSS 源
- 使用 RSSHub 等中转服务

### 数据持久化
所有数据存储在本地浏览器:
- 清除浏览器缓存会导致数据丢失
- 请定期导出备份
- 不同设备间需要手动导入

### 图片加载
部分图片可能有防盗链限制:
- 这是正常现象
- 不影响文字内容阅读

## 🆘 常见问题

### Q: 为什么有些 RSS 源无法添加？
A: 可能是以下原因:
1. 链接不是有效的 RSS/Atom 地址
2. 网站禁止跨域访问
3. 网络连接问题

### Q: 如何找到 RSS 源地址？
A: 大多数网站会在底部或侧边栏提供 RSS 订阅链接。也可以使用 RSSHub 等工具生成。

### Q: 数据会同步到云端吗？
A: 当前版本仅本地存储，暂不同步。请定期备份重要数据。

### Q: 可以在电脑上使用吗？
A: 可以！虽然是为移动端设计，但在电脑浏览器上也能正常使用。
