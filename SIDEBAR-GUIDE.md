# 🎨 侧边栏布局使用指南

## ✨ 新功能特性

### 📱 可折叠侧边栏

全新的界面布局，左侧为 RSS 源列表侧边栏，右侧为主内容区域。

**主要特点**:
- ✅ 侧边栏可折叠/展开
- ✅ 响应式设计，适配各种屏幕
- ✅ 移动端自动切换为抽屉式
- ✅ 优雅的过渡动画

## 📐 尺寸规格

### 桌面端 (>768px)

**侧边栏展开状态**:
- 宽度：280px
- 显示完整信息 (标题、文章数、操作按钮)

**侧边栏折叠状态**:
- 宽度：60px
- 仅显示图标和简短标题

### 移动端 (≤768px)

**默认状态**:
- 侧边栏隐藏
- 全屏显示内容

**展开状态**:
- 侧边栏覆盖全屏
- 带遮罩层
- 点击遮罩关闭

## 🎮 操作方式

### 折叠/展开侧边栏

**桌面端**:
```
点击侧边栏右上角的折叠按钮 ⟷
• 单击即可切换
• 平滑过渡动画
```

**移动端**:
```
从屏幕左侧边缘向右滑动
或点击菜单按钮 (如果有)
```

### 选择 RSS 源

**步骤**:
1. 在侧边栏中找到目标 RSS 源
2. 点击该源
3. 右侧自动显示对应文章列表
4. 当前选中的源会高亮显示

### 管理 RSS 源

**仅在侧边栏展开时可用**:

**刷新单个源**:
```
点击源卡片上的 🔄 图标
```

**删除源**:
```
点击源卡片上的 🗑️ 图标
确认删除
```

**添加新源**:
```
点击底部"添加 RSS 源"按钮
或 ➕ 图标
```

## ⚙️ 技术实现

### 组件结构

```vue
MainLayout (主布局组件)
├── Sidebar (侧边栏)
│   ├── Header (头部)
│   ├── Content (内容)
│   │   ├── EmptyState (空状态)
│   │   └── FeedsList (源列表)
│   └── Footer (底部)
└── MainContent (主内容区)
    └── slot (插槽)
```

### 核心代码

**折叠状态管理**:
```javascript
const sidebarCollapsed = ref(false)

function toggleSidebar() {
  sidebarCollapsed.value = !sidebarCollapsed.value
}
```

**响应式处理**:
```javascript
const isMobile = computed(() => window.innerWidth < 768)

watch(isMobile, (val) => {
  if (val) {
    sidebarCollapsed.value = true
  }
})
```

## 🎓 自定义扩展

### 修改侧边栏宽度

在 `MainLayout.vue` 中:

```scss
.sidebar {
  width: 280px; // 修改这里
  
  &.collapsed {
    width: 60px; // 折叠后宽度
  }
}
```

```scss
@media (max-width: 768px) { // 修改断点值
```

## 🐛 常见问题

### Q: 侧边栏不显示？

A: 检查:
1. 是否在主页面路由
2. 浏览器宽度是否太窄
3. 控制台是否有错误

### Q: 折叠按钮没反应？

A: 可能原因:
1. JavaScript 未加载完成
2. 事件监听器冲突
3. CSS 动画被禁用

### Q: 移动端无法滑动？

A: 确保:
1. 触摸事件正常
2. 没有阻止默认行为
3. 浏览器支持手势
