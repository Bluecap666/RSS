# 🚀 快速开始 - H5 RSS 阅读器

## ✅ 项目已就绪!

恭喜你!H5 RSS 阅读器已经创建完成并正在运行中。

## 📱 立即体验

**访问地址**: http://localhost:5174

点击预览按钮，在浏览器中打开应用!

## 🎯 第一次使用

### 1️⃣ 添加第一个 RSS 源

1. 打开应用后，你会看到"还没有添加 RSS 源"的提示
2. 点击蓝色的"**添加第一个 RSS 源**"按钮
3. 或者直接点击右下角的蓝色"+"按钮

### 2️⃣ 输入 RSS 地址

在输入框中输入以下任一示例地址:

```
# 知乎日报
https://rsshub.app/zhihu/daily

# 36 氪
https://rsshub.app/36kr/motif/实时

# 简书首页
https://rsshub.app/jianshu/home

# 少数派
https://rsshub.app/sspai/column
```

### 3️⃣ 等待解析

- 系统会自动解析 RSS 地址
- 显示源标题和描述预览
- 显示文章数量

### 4️⃣ 点击添加

确认无误后，点击右上角的"**添加**"按钮

## 📖 浏览文章

1. **返回首页**: 点击左上角返回箭头
2. **查看文章列表**: 点击刚才添加的 RSS 源卡片
3. **阅读文章**: 点击任意文章卡片

## ⚙️ 个性化设置

### 调整阅读体验

1. 打开任意文章
2. 点击右上角的**字体图标**
3. 调节以下内容:
   - **字体大小**: 拖动滑块 (14-24px)
   - **行间距**: 拖动滑块 (1.5-2.5)
   - **主题**: 选择日间/夜间/复古

### 备份你的数据

1. 返回首页
2. 点击右上角的**设置图标**
3. 点击"**导出 RSS 源**"
4. 保存 JSON 文件到安全位置

## 💡 更多功能

### 刷新 RSS 源

- 在 RSS 源卡片上点击**刷新图标**
- 或在文章列表页面点击右上角刷新按钮

### 删除 RSS 源

- 在 RSS 源卡片上点击**删除图标**
- 确认后即可删除

### 切换主题

在阅读页面:
- 点击底部主题选项
- 选择适合你的阅读模式

## 🔍 常见问题

### Q: 找不到 RSS 源地址？

**A**: 可以在以下地方找到:
- 网站底部或侧边栏
- 浏览器地址栏图标 (橙色 RSS 标志)
- 使用 RSSHub 生成

### Q: 解析失败怎么办？

**A**: 可能原因:
1. 网络连接问题
2. RSS 地址无效
3. 网站禁止跨域访问

**解决方案**:
- 检查网络
- 更换 RSS 源
- 使用 RSSHub 等中转服务

### Q: 如何找到更多 RSS 源？

**A**: 推荐资源:
- **RSSHub**: https://docs.rsshub.app
- **Feedly**: https://feedly.com
- **Inoreader**: https://inoreader.com
